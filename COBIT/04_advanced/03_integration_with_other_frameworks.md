## Overview

## Framework Integration

### ITIL Integration

- **COBIT:** Governance objectives
- **ITIL:** Service management practices
- **Together:** COBIT sets objectives, ITIL provides practices

### ISO/IEC 27001 Integration

- **COBIT:** IT governance
- **ISO 27001:** Information security management
- **Together:** COBIT provides governance, ISO 27001 provides security controls

### ISO/IEC 20000 Integration

- **COBIT:** IT governance
- **ISO 20000:** IT service management
- **Together:** COBIT provides governance, ISO 20000 provides service management

### NIST Integration

- **COBIT:** IT governance
- **NIST:** Cybersecurity framework
- **Together:** COBIT provides governance, NIST provides cybersecurity guidance
