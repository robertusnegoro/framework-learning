# COBIT 2019 Maturity Assessment

**Level:** Implementation  
**Framework:** COBIT 2019  
**Last Updated:** January 2025

---

## Overview

This guide covers how to assess governance maturity using COBIT 2019.

---

## Maturity Model

COBIT 2019 uses a maturity model with levels 0-5:

- **Level 0:** Incomplete
- **Level 1:** Performed
- **Level 2:** Managed
- **Level 3:** Established
- **Level 4:** Predictable
- **Level 5:** Optimizing

---

## Assessment Process

### Step 1: Plan Assessment

- Define scope
- Select objectives to assess
- Prepare assessment
- Allocate resources

### Step 2: Conduct Assessment

- Gather evidence
- Assess current maturity
- Document findings
- Validate with stakeholders

### Step 3: Analyze Results

- Analyze maturity levels
- Identify gaps
- Compare with targets
- Identify improvements

### Step 4: Report Findings

- Create assessment report
- Present findings
- Get feedback
- Plan improvements

---

## Key Considerations

- **Evidence-Based:** Use evidence for assessment
- **Stakeholder Input:** Involve stakeholders
- **Honest Assessment:** Be honest about current state
- **Actionable:** Provide actionable recommendations
