# COBIT 2019 Certification Paths

**Last Updated:** January 2025

---

## Overview

This document outlines COBIT 2019 certification paths and requirements.

---

## Certification Structure

### Entry Level

**COBIT 2019 Foundation**
- Prerequisites: None
- Exam: 75 questions, 120 minutes, 65% to pass
- Foundation for all other certifications

### Intermediate Level

**COBIT 2019 Design & Implementation**
- Prerequisites: COBIT 2019 Foundation
- Focus: Designing and implementing governance systems

**COBIT 2019 Assessor**
- Prerequisites: COBIT 2019 Foundation
- Focus: Assessing governance maturity

---

## COBIT 2019 Foundation

### Exam Information

- **Format:** Multiple choice
- **Questions:** 75
- **Duration:** 120 minutes
- **Passing Score:** 65% (49 out of 75)

### Topics Covered

- Framework introduction
- Governance vs management
- 40 objectives overview
- Design factors
- Implementation approach

## Recommended Paths

### Path 1: Foundation Only

1. COBIT 2019 Foundation
- Good for understanding COBIT basics

### Path 2: Implementation Focus

1. COBIT 2019 Foundation
2. COBIT 2019 Design & Implementation
- Good for implementing COBIT

### Path 3: Assessment Focus

1. COBIT 2019 Foundation
2. COBIT 2019 Assessor
- Good for assessing governance

### Path 4: Complete Path

1. COBIT 2019 Foundation
2. COBIT 2019 Design & Implementation
3. COBIT 2019 Assessor
- Complete COBIT certification
