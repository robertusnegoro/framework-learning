## Exam Overview

### Exam Format

- **Type:** Multiple choice
- **Questions:** 40 questions
- **Duration:** 60 minutes
- **Passing Score:** 26 out of 40 (65%)
- **Language:** English (and other languages)
- **Format:** Closed book

### Exam Topics

The exam covers:

1. **Service Value System (SVS)**
   - Components of SVS
   - How components work together

2. **Service Value Chain**
   - Six activities
   - How activities create value

3. **Guiding Principles**
   - Seven principles
   - Application of principles

4. **Four Dimensions**
   - All four dimensions
   - How dimensions work together

5. **Practices**
   - Key practices (Incident, Problem, Change, etc.)
   - Practice purposes and key concepts

## Key Topics to Study

### 1. Service Value System (SVS)

**Must Know:**
- Five components: Guiding Principles, Governance, Service Value Chain, Practices, Continual Improvement
- How components work together
- Input (Opportunity/Demand) → SVS → Output (Value)

**Practice Questions:**
- What are the components of the SVS?
- How does the SVS create value?
- What is the relationship between components?

### 2. Service Value Chain

**Must Know:**
- Six activities: Plan, Improve, Engage, Design & Transition, Obtain/Build, Deliver & Support
- Purpose of each activity
- How activities work together
- Value streams

**Practice Questions:**
- What are the six Service Value Chain activities?
- What is the purpose of "Engage"?
- How do activities create value streams?

### 3. Guiding Principles

**Must Know:**
- All seven principles
- Definition of each principle
- How to apply principles
- Examples of each principle

**Practice Questions:**
- What are the seven guiding principles?
- What does "Start Where You Are" mean?
- Give an example of "Focus on Value"

### 4. Four Dimensions

**Must Know:**
- All four dimensions
- Key aspects of each dimension
- Why all dimensions must be considered
- How dimensions work together

**Practice Questions:**
- What are the four dimensions?
- What does "Organizations & People" include?
- Why are four dimensions important?

### 5. Practices

**Must Know:**
- 34 practices (know categories and key practices)
- Key practices: Incident, Problem, Change, Service Level, Service Request, Continual Improvement
- Purpose of key practices
- Key concepts for each practice

**Practice Questions:**
- What is the purpose of Incident Management?
- What is the difference between Incident and Problem?
- What are change types?

## Practice Exam Tips

### Reading Questions

1. **Read Carefully**
   - Read entire question
   - Understand what's being asked
   - Watch for negative words ("not", "except")

2. **Identify Key Words**
   - Look for ITIL-specific terms
   - Identify the topic
   - Note the question type

### Answering Questions

1. **Eliminate Wrong Answers**
   - Cross out obviously wrong answers
   - Narrow down choices
   - Use process of elimination

2. **Look for Best Answer**
   - Multiple answers may seem correct
   - Choose the BEST answer
   - Consider ITIL principles

3. **Don't Overthink**
   - Trust your knowledge
   - Don't second-guess
   - Answer and move on

### Time Management

1. **Pace Yourself**
   - 60 minutes for 40 questions
   - ~1.5 minutes per question
   - Don't spend too long on one question

2. **Flag Difficult Questions**
   - Answer if you know it
   - Flag if unsure
   - Return to flagged questions

3. **Review Time**
   - Leave time for review
   - Check flagged questions
   - Verify answers
