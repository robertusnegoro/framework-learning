# Real-Time Decision Making

**Level:** Intermediate  
**Framework:** FinOps Foundation  
**Last Updated:** January 2025

---

## Overview

This capability focuses on enabling real-time decision making about cloud spending.

---

## Key Activities

### Provide Real-Time Data

Provide data in real-time or near real-time.

**Data Types:**
- Cost data
- Usage data
- Performance data
- Anomaly alerts

### Enable Decision-Making

Enable teams to make decisions quickly.

**Decision Support:**
- Decision frameworks
- Decision guidelines
- Decision tools
- Decision processes

### Set Up Alerts

Set up alerts for anomalies and thresholds.

**Alert Types:**
- Cost anomalies
- Budget alerts
- Usage alerts
- Performance alerts

---

## Maturity Levels

### Crawl: Weekly Decisions

- Weekly data
- Weekly decisions
- Manual alerts
- Basic decision support

### Walk: Daily Decisions

- Daily data
- Daily decisions
- Automated alerts
- Good decision support

### Run: Real-Time Decisions

- Real-time data
- Real-time decisions
- Real-time alerts
- Excellent decision support

---

## Decision Frameworks

### Cost-Benefit Analysis

Analyze costs vs benefits.

### Unit Economics

Understand cost per unit of value.

### Trade-Offs

Balance speed, cost, and quality.

### Value-Driven Decisions

Make decisions based on business value.
