# Cloud Rate Optimization

**Level:** Intermediate  
**Framework:** FinOps Foundation  
**Last Updated:** January 2025

---

## Overview

This capability focuses on optimizing the rates paid for cloud resources.

---

## Key Activities

### Negotiate Discounts

Negotiate discounts with cloud providers.

**Discount Types:**
- Volume discounts
- Enterprise discounts
- Commitment discounts

### Use Reserved Instances / Savings Plans

Use commitment-based discounts.

**AWS:**
- Reserved Instances
- Savings Plans

**Azure:**
- Reserved Capacity
- Azure Savings Plans

**GCP:**
- Committed Use Discounts

### Use Spot Instances

Use spare capacity at lower prices.

**Benefits:**
- Significant cost savings
- For flexible workloads
- Requires automation

### Optimize Commitments

Optimize reserved instance and savings plan commitments.

**Optimization:**
- Right-size commitments
- Optimize commitment mix
- Manage commitment lifecycle
